# AI 3개 툴 수동 통합 테스트 체크리스트

## 대상 페이지
- `/ai-id-photo`
- `/ai-passport-photo`
- `/ai-photoshop` (프로필 실행 페이지)

## 사전 조건
- 프론트: `npm run dev` (기본 `http://localhost:4321`)
- Node API: `cd node-api && npm run dev` (기본 `http://127.0.0.1:8787`)
- `.env` 권장
  - `EXTERNAL_AI_ENABLED=false`
  - `PAYMENT_MODE=mock`

## 공통 기능 테스트 (각 페이지별 반복)
1. 페이지 진입 후 상태문구 확인
- `API 연결됨` 또는 `API 미연결 · 로컬 fallback 모드`

2. 업로드 테스트
- 정상 이미지 업로드
- 결과: 미리보기 노출 + 생성 버튼 활성화
- 품질문구: 저해상도/얼굴 감지 힌트가 나오면 정상

3. 카메라 테스트
- 카메라 시작 > 촬영
- 결과: 미리보기 반영 + 품질문구 출력 가능

4. 기본 생성 테스트
- `...만들기` 클릭
- 결과: 후보 3장 생성 + 카드 선택 가능

5. 선택/다운로드 테스트
- 후보 2장 선택 후 다운로드
- 결과: 파일 2개 다운로드 시작

6. 추가 생성 테스트
- `+2` 클릭
- 결과: 후보 수 증가
- API 미연결이면 로컬 추가 생성 안내문구 표시

7. 새로고침 복원 테스트
- 후보 생성 후 `F5`
- 결과: 이전 후보 복원 문구 + 후보 유지

8. 초기화 테스트
- `초기화` 클릭
- 결과: 미리보기/후보/복원 상태 초기화

## SEO/링크 테스트
1. 교차 이동 링크
- 증명 -> 여권/프로필 이동
- 여권 -> 증명/프로필 이동
- 프로필 -> 증명/여권 이동

2. SEO 랜딩 링크
- `/ai-headshot` -> 실행 페이지 이동 확인
- `/online-id-photo`, `/online-passport-photo` 등 연결 확인

## 합격 기준
- 3개 페이지 모두 업로드/촬영/기본생성/추가생성/다운로드/복원 동작
- 콘솔 치명 에러 없음
- API 미연결 상태에서도 로컬 fallback 다운로드 가능
