현재 프로젝트 AI 증명사진 기능 분석에 대한 검토 메모

검토 결론
- 기존 분석 내용의 핵심 판단은 맞음
- 현재 구현은 기획과 일치하지 않음
- 지금 상태는 결제/UX 프로토타입 + 단순 이미지 변형 수준으로 보는 것이 정확함

맞는 점
- "지저분한 셀피 -> 취업 가능한 정장 증명사진" 파이프라인으로 동작하지 않는다는 판단은 맞음
- 현재 생성 로직이 원본 리사이즈/밝기보정/약간 다른 크롭 수준이라는 판단도 맞음
- 새 얼굴을 생성하지 않는다는 판단도 맞음
- 얼굴 detection 기반 정렬, 리터칭, 정장 합성, 규격 크롭이 빠져 있다는 지적도 맞음

정리된 실무 판단
- 지금은 도메인 배포 단계가 아님
- 지금은 실제 AI 증명사진 엔진이 아니라 프로토타입 단계임
- 기획 핵심 단계인 아래 항목이 아직 미구현 상태임
  1) 얼굴 정렬
  2) 얼굴 동일성 보호
  3) 피부/얼굴 리터칭
  4) 헤어 정리
  5) 정장 합성
  6) 배경 교체의 실제 활성화
  7) 규격 기반 크롭

현재 상태를 가장 정확히 표현하면
- 결제/복원/후보 선택 UX는 어느 정도 구현됨
- 실제 사진 품질 파이프라인은 미완성임
- 따라서 "취업 가능한 정장 증명사진" 서비스 품질에는 아직 도달하지 못함

권장 우선순위
1) 외부 배경 제거 실제 연결
2) 서버 기준 얼굴 정렬 + 규격 크롭
3) 비생성 리터칭
4) 정장 합성
5) 얼굴 동일성 검증

최종 메모
- 기존 분석은 방향이 맞음
- 오타 1개와 외부 AI 사용 상태 표현만 다듬으면 현재 프로젝트 상태를 정확히 설명하는 문서로 사용할 수 있음
