# 증명사진 홍보 설명란 기획 (페이지 버튼 하단)

## 목적
- `증명사진 만들기` 버튼 아래에서 신뢰/전환을 높이는 홍보 구간 제공
- 사용자가 결과 품질을 직관적으로 이해하도록 `샘플 변화 흐름`을 먼저 보여줌

## 위치
- 페이지: `/ai-id-photo`
- 위치: 1) 사진 입력 카드 내부, `증명사진 만들기` 버튼 바로 아래

## 섹션 구성
1. 제목: 셀카가 증명사진으로 바뀌는 방식
2. 안내 문구: 샘플 화면이며 실제 결과는 원본 상태에 따라 다를 수 있음
3. 3단계 카드
- Before: 일반 셀카
- Step: 배경 정리 + 얼굴 정렬
- After: 증명사진 규격 결과
4. 처리 단계 요약
- 정면 얼굴 자동 감지
- 흰 배경/규격 비율 자동 적용
- 밝기/톤 미세 보정 후보 3장 생성

## 운영 문구
- AI 생성 결과는 원본 사진 상태에 따라 달라질 수 있습니다.
- 다운로드 후 제출 전 각 기관의 사진 규격을 확인하세요.

## 확장 계획
- 실제 샘플 이미지 파일(`public/examples/`) 연결
- 슬라이더 비교 컴포넌트 추가(정적 샘플 전용)
- 여권사진/AI 포토샵 페이지에도 동일 패턴 적용
