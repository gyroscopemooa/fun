# AI 사진 입력 품질 처리 규칙 (CLI 판단용)

## 목적

사용자가 업로드하는 다양한 사진 상태에 대해 AI가 자동으로 `증명사진 / 여권사진 / 프로필사진` 생성 가능 여부를 판단하고 처리한다.

## 1. 지원 사진 유형

다음 유형의 사진을 입력으로 받을 수 있다.

1. 정면 얼굴 사진
2. 전신 사진
3. 셀카 사진
4. 저해상도 사진
5. 방송/스크린샷 캡처 사진

AI는 업로드된 사진을 자동 분석하여 가능한 경우 증명사진 스타일로 변환한다.

## 2. 자동 얼굴 중심 처리

사진에 얼굴이 포함된 경우 AI는 다음 작업을 수행한다.

1. 얼굴 감지
2. 얼굴 중심 크롭
3. 얼굴 정렬
4. 배경 제거
5. 증명사진 규격 크롭

전신 사진이라도 얼굴이 인식되면 얼굴 중심으로 자동 재구성한다.

## 3. 저품질 사진 처리

다음 경우 자동 보정을 시도한다.

- 저해상도
- 노이즈
- 어두운 조명
- 스크린샷
- 셀카 필터

AI는 가능한 범위에서 아래를 수행한다.

- 업스케일
- 노이즈 제거
- 밝기 보정
- 얼굴 복원

## 4. 처리 실패 조건

다음 경우 증명사진 생성이 어려울 수 있다.

- 얼굴 미인식
- 얼굴 가림
- 극단적 측면 얼굴
- 해상도 매우 낮음

이 경우 메시지를 표시한다.

`얼굴을 인식할 수 없습니다. 다른 사진을 업로드해주세요.`

## 5. 자동 규격 생성

사용자가 선택한 규격에 따라 자동 생성한다.

지원 규격:

- 한국 증명사진
- 여권사진
- 프로필 사진

AI는 다음 요소를 자동 적용한다.

- 얼굴 비율
- 배경 색상
- 크기
- 크롭

## 6. 결과 생성 방식

AI 처리 후 결과 이미지를 생성한다.

결과는 다음 조건을 만족해야 한다.

- 정면 얼굴 중심
- 깨끗한 배경
- 적절한 얼굴 비율
- 증명사진 규격

## 7. 사용자 안내 문구

생성 결과 아래에 표시한다.

`AI 생성 결과는 원본 사진 상태에 따라 달라질 수 있습니다.`

`다운로드 후 제출 전, 각 기관의 사진 규격을 확인하시기 바랍니다.`
